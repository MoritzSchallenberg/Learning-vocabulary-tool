# 📚 Learning Vocabulary Tool
Ein einfaches Tool zum Lernen und Verwalten von Vokabeln.  
Das Projekt ermöglicht es, neue Wörter zu speichern, zu üben und den eigenen Lernfortschritt zu verbessern.


## 🚀 Projektübersicht
Dieses Projekt wurde entwickelt, um das Lernen von Vokabeln zu vereinfachen.  
Nutzer können:
- neue Wörter hinzufügen
- Bedeutungen speichern
- Vokabeln üben
- ihren Lernfortschritt verbessern
Das Tool eignet sich besonders für:
- 📖 Sprachlernen (z. B. Englisch, Deutsch)
- 🎓 Studium & Schule
- 🧠 eigenständiges Lernen


## 🧠 Funktionsprinzip
Das System basiert auf einer einfachen Struktur:
- Wörter werden gespeichert
- beim Üben werden diese zufällig abgefragt
- der Nutzer gibt die Übersetzung ein oder überprüft sie


## ⚙️ Features
- Vokabeln speichern
- einfache Abfrage / Quiz-System
- lokale Speicherung der Wörter
- übersichtliche Struktur


## 🛠️ Technologien
- Programmiersprache: (z. B. Python / JavaScript – anpassen!)
- Speicherung: lokal (Dateien / einfache Datenstruktur)
- optional: Web-Technologien für UI
